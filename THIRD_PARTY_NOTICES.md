# Third-Party Notices

This repository contains Nucleomic source code.

Third-party tools such as **MUSCLE**, **ClustalW**, and **IQ-TREE** are runtime dependencies and should preferably be installed separately rather than committed into this repository as bundled binaries.

Before redistributing any bundled third-party executable, verify the license terms and redistribution conditions for the exact version you ship.

Suggested approach for this repository:
- keep third-party binaries out of Git
- document installation steps in the README or a dedicated dependency guide
- preserve upstream copyright and license notices where required
